코일스프링(인장, 압축)계산식

　

      

　

  

　

　

                            ■ 계산기호

 d  : 스프링 선경 (㎜)

 D  : 코일 평균경 (㎜)

 D1 : 코일 내경 (㎜)

 D2 : 코일 외경 (㎜)

 Na : 유효 감김수 

 G : 횡탄성 계수 (Kgf/㎣)

　

 C : 스프링 지수

　

 P : 작용하중 (Kgf)

 Pi : 초기장력 (Kgf)

  δ : 스프링의 변위량 (㎜)

  K : 스프링 상수 (Kgf/㎜)

  Kb : 응력 수정 계수

  ※ Kb =

  τt : 비틀림 응력 (Kgf/㎣)

  τb : 비틀림 수정 응력 (Kgf/㎣)

  τi : 초기장력에 의한 비틀림 응력 (Kgf/㎣)

　

　

                             ■ 주요계산식 (압축, 인장스프링)

* 압축스프링 및 초기장력이 있는 인장 스프링의 경우 ( 단, P > Pi )

  

   초기장력이 없는 인장스프링의 경우에는

   Pi = O 로 하여 계산식에 적용함.
